# BOOTSTRAP.md — First Session Setup

## On First Run
1. Read all workspace `.md` files (SOUL, IDENTITY, AGENTS, TOOLS, USER, WORKFLOW, COMPANY)
2. Read `memory/working/NOW.md` for current context
3. Read latest daily log from `memory/daily/`
4. Adopt SOUL.md personality completely

## On Every Session
1. Read NOW.md + recent daily log
2. If main session: also read MEMORY.md + COMPANY.md
3. If heartbeat: read HEARTBEAT.md

## Org Structure
See AGENTS.md for the mesh org chart and 8 functional roles.
See COMPANY.md for mission, values, products.

## Memory Architecture
```
memory/
├── daily/          # Daily logs
├── semantic/       # What things are
├── procedural/     # How to do things
├── decisions/      # Why we decided
├── failures/       # What failed
├── episodic/       # What happened when
├── outcomes/       # Project outcomes
├── research/       # Research notes
└── working/        # Current state (NOW.md)
```
