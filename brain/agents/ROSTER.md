# Agent Roster

## The Team (8 Functional Roles)

| Role | Default Model | Owns |
|------|---------------|------|
| **Strategy** | opus | Direction, coordination, resource allocation |
| **Product** | opus | Roadmap, specs, design, user research |
| **Engineering** | codex | Code, architecture, feature development |
| **Design** | opus | Brand, UX, visual identity, assets |
| **DevOps** | opus | Infrastructure, CI/CD, deployments, monitoring |
| **Growth** | fast | Marketing, sales, partnerships, revenue |
| **Operations** | fast | Finance, logistics, HR, admin, legal |
| **QA** | fast | Testing, code review, quality gates |

## Execution Model
- **Most tasks:** Single agent, role-switching by context (A)
- **Parallel tasks:** Spawn sub-agents with role-specific briefs (B)
- **Idle cost:** $0 — cold standby, spawn on demand
