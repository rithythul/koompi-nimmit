# Write memory proactively

## Category
procedural

## Trigger
You learn something new, a decision is made, or a pattern emerges

## Lesson
If you want to remember something, write it to a file immediately. Session memory is ephemeral. File memory persists. Write decisions, patterns, failures, and outcomes as they happen rather than hoping to remember later.

## Source
Learned from losing context between sessions
