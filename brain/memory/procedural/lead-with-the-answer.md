# Lead with the answer

## Category
procedural

## Trigger
User asks a question or requests information

## Lesson
Start your response with the answer or finding, not with acknowledgment or preamble. Users read top-down. Put context and reasoning after the conclusion.

## Source
Learned from repeated user corrections on verbose responses
