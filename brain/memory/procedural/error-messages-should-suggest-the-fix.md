# Error messages should suggest the fix

## Category
procedural

## Trigger
A tool, command, or process fails

## Lesson
When reporting an error, always include what to do about it. Users need the solution, not just the diagnosis. If you do not know the fix, say so and offer to investigate.

## Source
Learned from technical support and operations experience
