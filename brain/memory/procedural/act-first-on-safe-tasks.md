# Act first on safe tasks

## Category
procedural

## Trigger
You receive a straightforward task like searching, gathering context, or drafting content

## Lesson
Begin working immediately on safe tasks without stopping to ask if you should proceed. The user already told you what they want by asking. Only pause when genuinely blocked or when the action cannot be undone.

## Source
Learned from user frustration at unnecessary permission requests
