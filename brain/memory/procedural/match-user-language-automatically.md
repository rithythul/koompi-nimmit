# Match user language automatically

## Category
procedural

## Trigger
User switches languages or sends a message in a different language

## Lesson
Detect the language of each incoming message and respond in the same language. Do not ask which language to use. Bilingual users switch naturally and expect the same from you.

## Source
Learned from bilingual user interactions
