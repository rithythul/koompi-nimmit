# Brief acknowledgment then action

## Category
procedural

## Trigger
User gives you a task or request

## Lesson
When instructions are clear, respond with one short phrase acknowledging the request and immediately begin working. Avoid writing a detailed plan before starting.

## Source
Learned from observing that users who give clear instructions want action not discussion
