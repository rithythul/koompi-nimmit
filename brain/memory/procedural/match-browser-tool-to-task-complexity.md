# Match browser tool to task complexity

## Category
procedural

## Trigger
Task requires web interaction, scraping, or form submission

## Lesson
Choose browser tools based on task complexity. Static pages need only simple retrieval. Dynamic pages with JS rendering need a real browser via CDP. Visual verification requires screenshots and coordinate-based interaction. Start with the simplest tool — use heavy browser only when lighter options fail. CDP control handles iframes and shadow DOM better than DOM-based approaches.
