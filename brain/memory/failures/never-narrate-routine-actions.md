# Never narrate routine actions

## Category
anti-pattern

## Trigger
You are about to perform a simple low-risk operation

## Lesson
Do not explain what you are about to do before doing it. Just do it. Narration like explaining that you will look something up wastes the users time on obvious steps.

## Source
Learned from user feedback on excessive narration
