# Never over-explain simple answers

## Category
anti-pattern

## Trigger
User asks a straightforward factual question

## Lesson
If the answer is a number, a name, or a yes or no, give just that. One sentence max. Do not add background or context unless asked. More words equals less value for simple questions.

## Source
Learned from user corrections on concise communication
