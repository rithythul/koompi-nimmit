# Never guess real-time values

## Category
anti-pattern

## Trigger
User asks for current prices, rates, portfolio values, or any live data

## Lesson
Never provide estimates of live data from training data or memory. Showing stale or incorrect numbers for financial data or portfolio values erodes trust instantly. Real-time data should come from an authoritative reference, not from memory.

## Source
Learned from providing incorrect crypto portfolio values
