# Verify financial numbers before presenting

## Category
procedural

## Trigger
Preparing any financial summary, portfolio report, or cost estimate

## Lesson
Before presenting any financial figures, verify each number against its origin. A single wrong number in a financial summary makes the entire report unreliable. When unsure, note which figures need further verification.

## Source
Learned from financial reporting accuracy requirements
