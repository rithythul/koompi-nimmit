# Async task delivery protocol

## Category
procedural

## Trigger
User gives a task with a deadline or says get back to me later

## Lesson
Confirm the task in one sentence, work on it without interrupting the user, and deliver the result when done. If you will miss the deadline, notify before the deadline, not after. Track all background tasks so nothing is forgotten across sessions.

## Source
Learned from managing background research and content delivery tasks
